# Study 016 – Eye Mind Wander Database: Complete Dataset Details

## Details

**Eye Tracker:** Webcam-based system using WebGazer technology

**Participant Recruitment:** Prolific

**Task Description:** Participants selected necessary options to solve mathematical problems, then provided answers

**Task Classification:** Other

**Performance Measure:** Incorrect actions

**Mind Wandering Assessment:**
- TUT (Thought-Unrelated-to-Task)
- Disengagement
- Valence
- Boredom

**Gaze Data:**
- Gaze fixations recorded
- Offscreen gaze instances captured
- Measurements averaged across entire problem-solving period

**Areas of Interest:** N/A
